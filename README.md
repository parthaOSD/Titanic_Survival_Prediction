## Titanic Survival Prediction

## Overview
   This project demonstrates the use of machine learning technique to predict the whether a passenger survived in the Titanic Disaster. The model is trained on passenger data and 
   evaluated using training and testing accuracy scores.

## Dataset Used
   Kaggle Titanic Dataset

## Objective
  The objective of this project is to build a machine learning classification model that predicts whether a passenger survived or not in Titanic based on passenger-related features. 
  The model is trained and evaluated using train and test datasets, and its performance is measured using accuracy scores.

## Machine Learning Model Used
   Logistic Regression Model

## Data Preprocessing
   Missing values in the dataset were handled appropriately using mean imputation to improve model performance.

## Evaluation Metrics
   1) train accuracy
   2) test accuracy

## Results
   By using Logistic Regression, model achieved training and testing accuracy which indicates its performance in predicting
   whether a passenger survived the Titanic disaster.

## Technologies Used
   1) Python
   2) Numpy
   3) Pandas
   4) Seaborn
